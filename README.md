PayPal Payment for Laravel
====================

LaravelでPayPal決済を実行するサンプルコードです。


Requirement
--------------------

* git
* bash
* docker
* PayPalビジネスアカウント


How To Use
--------------------

1. Git clone & Change directory
1. Excecute sail comannd for setup Docker Container

```bash
# docker containerを起動(初回起動なのでbuildも実行される)
vendor/bin/sail up -d
```

--------------------
